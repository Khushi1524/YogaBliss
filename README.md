Yoga Bliss is a wellness-focused website designed to promote yoga and meditation through a calm and visually soothing interface.
The project focuses on creating a peaceful user experience with well-structured content, smooth layout, and a clean design that reflects the essence of mindfulness and relaxation.

Features :

1. Clean and calming UI design
2. Structured sections for yoga and meditation
3. User-friendly layout
4. Responsive design
5. Smooth browsing experience

Tech Stack :

1. HTML
2. CSS
3. JavaScript
4. PHP 

Inspiration :
The design focuses on simplicity, calm colors, and balance to reflect the true essence of yoga and meditation.
